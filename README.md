# 수질오염총량검토서 작성 시스템 v3

## v3 변경사항

### 사업개요 탭
- **계산 파라미터 박스 제거**
- **사업위치**: 광역/기초 드롭박스 + 상세주소 입력 (인구수 원단위 시트 파싱)
- **용도지역**: 농림지역 추가 (세부 없음)
- **시가화/비시가화**: UI 제거 → 용도지역 대분류로 자동 판단
  - 주거·상업·공업 → 시가화
  - 녹지·관리·농림·자연보전 → 비시가화
- **세대당 인구수**: UI 제거 → DB 자동 조회

### 생활계 탭
- **처리방법**: 동별 3단계 cascade 드롭박스
  - 1단계: 공공하수처리시설 / 개인하수처리시설 / 정화조
  - 2단계(공공): 기초자치단체 기준 처리장 목록 (data.xlsx 하수처리장 시트)
  - 2단계(개인): 고도처리 / 일반처리
  - 3단계(개인): 50톤 이상 / 50톤 미만
  - 정화조: 추가 선택 없음

### 방류수질 기준
| 처리방법 | 조건 | BOD | T-P |
|---------|------|-----|-----|
| 공공하수처리시설 | - | data.xlsx 기재값 | data.xlsx 기재값 |
| 개인하수처리시설 | 50톤 이상 | 10 | 2 |
| 개인하수처리시설 | 50톤 미만 | 20 | 4 |
| 개인하수처리시설 | 수변구역 해당 (규모무관) | 10 | 2 |
| 정화조 | - | 발생부하량 × 0.75 (BOD 25% 삭감) ※확인 필요 | 변화없음 |

### ⚠️ 정화조 방류수질
현재 "BOD 25% 삭감" 적용 중. 실제 기술지침 기준 확인 후 `calc.js` 내 주석 위치 수정 필요.
